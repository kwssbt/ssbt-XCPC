# 最短路

默认 n 个点的编号为 0 ~ n-1

## Floyd

- 基于邻接矩阵

- 得到图中**任意两点间**的最点距离

- 可处理负边，可检查负环

- 时间复杂度: $O(N^3)$

```cpp
const int N=105; // 点的数量

int dis[N][N]; // dis[u][v] = u->v的距离

const int INF=0x3f3f3f3f; // 无穷远

// 初始化
void init(int n){
    for(int i=0;i<n;++i){
        for(int j=0;j<n;++j){
            if(i==j)dis[i][j]=0;
            else dis[i][j]=INF; // 不可达
        }
    }
}
void floyd(int n){
    for(int k=0;k<n;++k){
        for(int i=0;i<n;++i){
            // 剪枝
            if(i==k||dis[i][k]==INF)continue;
            
            for(int j=0;j<n;++j){
                // 剪枝
                if(j==k||dis[k][j]==INF)continue;
                
                if(dis[i][j]>dis[i][k]+dis[k][j]){
                    dis[i][j]=dis[i][k]+dis[k][j];
                }
            }
        }
    }
}
bool check_floyd(int n){
    floyd(n);
    for(int i=0;i<n;++i){
        if(dis[i][i]<0){
            return 1; // 有负环
        }
    }
    return 0; // 无负环
}
```



## Dijkstra

- 给定起点，得到起点到每个点的最短距离

- 不能处理负边

- 时间复杂度 $O(MlogM)$

```cpp
// dis[v] = 起点 -> v 的最短距离
int dis[N]; 

// vis[u]=1 : 起点到 u 在最短距离已结算
bool vis[N];

// 初始化
void init(int n){
    for(int i=0;i<n;++i){
        dis[i]=INF;
        vis[i]=0;
    }
}
void dijkstra(int start){
    // 堆里存 { cost, u } :起点 -> u 的距离为 cost
    priority_queue<pair<int,int>,vector<pair<int,int>>,greater<>>pq;

    dis[start]=0;
    pq.push({0,start});
    
    while(pq.size()){
        auto [c,u]=pq.top();
        pq.pop();

        if(vis[u])continue;
        vis[u]=1;
        
        for(int ei=head[u];ei!=-1;ei=nex[ei]){
            int v=to[ei];
            int w=weight[ei];
            int nc=c+w;
            if(dis[v]>nc){
                dis[v]=nc;
                pq.push({nc,v});
            }
        }
    }
}
```

## SPFA

- 给定起点，得到起点到每个点的最短距离

- 可以处理负边，可以检查负环

- 时间复杂度 $O(NM)$ ，但在随机图上常数较好

```cpp
// dis[v] = 起点 -> v 的最短距离
int dis[N]; 

// in[u]=1 ：点 u 在队列中
bool in[N]; 
 
void init(int n){
    for(int i=0;i<n;++i){
        in[i]=0;
        dis[i]=INF;
    }
}
void spfa(int start){
    // 队列存可能松弛的点
    queue<int>q;
    
    dis[start]=0;
    q.push(start);
    in[start]=1;
    
    while(q.size()){
        int u=q.front();
        q.pop();
        in[u]=0;
        for(int ei=head[u];ei!=-1;ei=nex[ei]){
            int v=to[ei];
            int w=weight[ei];
            if(dis[v]>dis[u]+w){
                dis[v]=dis[u]+w;
                if(in[v]==0){
                    q.push(v);
                    in[v]=1;
                }
            }
        }
    }
}
```

**检查负环**

$n$ 个点

记录最短路树上的边数

一条不包含负环的最短路径，最多只能包含 $n-1$ 条边

若有规定起点 $s$ ，就以 $s$ 为起点

若没有规定起点，添加一个 **虚拟起点 S**

**虚拟起点 S** 连接其它所有的节点，边权为 **0**

```cpp
int cnt[N] // cnt[u] : 从起点到 u 的最短路上的边数

bool spfa_check(int s, int n){
    for(int i=0;i<n;++i){
        dis[i]=INF;
        in[i]=0;
        cnt[i]=0;
    }

    queue<int> q;
    q.push(s);
    in[s]=1;
    cnt[s]=0;
    dis[s]=0;
    
    while(q.size()){
        int u=q.front();
        q.pop();
        in[u] = 0;
        for(int ei=head[u];ei!=-1;ei=nex[ei]){
            int v=to[ei];
            int w=weight[ei];
            if(dis[v]>dis[u]+w){
                dis[v]=dis[u]+w;
                cnt[v]=cnt[u]+1;
                
                if(cnt[v]>=n)return 1; // 经过的边数达到 n，必定有负环
    
                if(in[v]==0){
                    q.push(v);
                    in[v]=1;
                }
            }
        }
    }
    return 0; // 无负环
}
```
